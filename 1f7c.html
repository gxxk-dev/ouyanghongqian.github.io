搭建能在网页玩的MC

前言

最近小区被封了，我便多出来了一大段时间玩电脑（毕竟不用上学）
然后我在无意间看到了 Eaglecraft 这个项目（详见MineBBS介绍帖），赶紧把项目Clone下来，接着就开始折腾文件

教程

  1. 下载页面https://ouyanghongqian.top/games/game/minecraft
  2. 登录服务器（建议使用SSH）
  3. 将本地所有文件上传至服务器，位置一定得是/var/www/html（如使用的软件路径与本文不一样，可自行替换）
  4. 执行代码：sudo apt install apache2 和 sudo service apache2 start
  5. 大功告成，现在可以通过服务器IP访问了（同时还能）

    注释

Eaglecraft. 作者LAX1DUDE，网页版MC的核心，基于MC1.5.2反编译而来，IMC.RE的网页服务就基于此项目，由于微软的一些操作，导致Github原仓库地址404，本文提供版本为Github其他用户所Fork的版本 ↩

2022-12-9更新：经测试，本文使用的版本过老，现已更新至原作者在2022-10-25发布的最终版本Eaglecraft22w43a（目前我得到的版本各功能均可正常使用，但文件疑似被更改过，如谁有可通过效验的原版本请务必在页面下留言），页面上方导航栏添加入口，可在线游玩
